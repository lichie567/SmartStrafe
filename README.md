# SmartStrafe

![release status](https://github.com/lichie567/SmartStrafe/actions/workflows/release.yml/badge.svg)
![build status](https://github.com/lichie567/SmartStrafe/actions/workflows/build-debug.yml/badge.svg?branch=main)

This plugin improves strafing when backpedaling with legacy movement controls.
Code for this plugin was borrowed from the SmartStrafe tweak in the SimpleTweaks plugin, authored by Iryoku. I created this plugin because I don't want to have to wait for all of SimpleTweaks to be updated on patch days.

Code for the original tweak can be found here: https://github.com/Caraxi/SimpleTweaksPlugin

## How to Install

SmartStrafe is not available in the standard Dalamud plugin repository and must be installed from my third party repository.

Here is the URL for my plugin repository: `https://raw.githubusercontent.com/lichie567/SmartStrafe/main/repo.json`
